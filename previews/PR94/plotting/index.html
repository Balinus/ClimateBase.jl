Python based plotting with cartopy

For now, you can use PyCall.jl, matplotlib, and the Python library cartopy. In the file ClimateBase/plotting/python.jl we provide two functions that plot maps of ClimArray in arbitrary projections: earthsurface for OrthogonalSpace and earthscatter for CoordinateSpace. You can incorporate these in your source code as a temporary solution.
